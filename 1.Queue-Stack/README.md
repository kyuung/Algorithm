# Array

-   Access = O(1)

<br/>
-   Array insertion

    -   [x, 1, 2, 3, 4, 5, 6] O(n)

<br/>
-   Array deletion

    -   [5, 2, 5, 3]

<br/>
-   Search O(n)

    -   [1, 2, 3, 6, 2, 4, 7] => O(n)
    -   binary serach => O (logN)

<br/>

### 장점

---

-   인덱스로 접근 가능하다.
-   다양한 데이터 타입을 담을 수 있다.
-   메모리 -> 데이터를 가져올건데 -> 옆에 있는것도 같이 가져옴 -> 그래서 빨라요
-   CPU <-- Access --> Cache <-- 엄청난 시간 ! --> Memory
